# Yew app

## Dependencies

install these globally `cargo`, `trunk`, `tailwindcss`

## Run app

Serve app
`trunk serve`

Tailwind watcher
`tailwindcss -w -o ./styles/output.css`

## Production deploy

`trunk build --production`
`tailwindcss -o ./styles/output.css --minify`
